#HR Analysis Power BI Dashboard

📊 Project Overview

The HR Analysis Power BI Dashboard is designed to analyze employee data and provide insights into workforce trends such as employee attrition, department performance, gender distribution, and job roles. This dashboard helps HR managers make data-driven decisions to improve employee management and organizational performance.

🎯 Objectives

1.Analyze employee attrition and retention trends

2.Understand workforce distribution by department, age, gender, and job role

3.Identify key factors affecting employee turnover

4.Provide interactive visual insights for HR decision-making

📁 Dataset Description

The dataset used in this project includes HR-related information such as:

1.Employee ID

2.Age

3.Gender

4.Department

5.Job Role

6.Education Level

7.Salary / Monthly Income

8.Years at Company

9.Attrition Status

📌 Key Dashboard Features

1.Employee Count Overview – Total employees, active employees, and attrition rate

2.Department-wise Analysis – Employee distribution across departments

3.Attrition Analysis – Attrition trends by age, department, and job role

4.Gender Distribution – Male vs Female employee analysis

5.Salary Insights – Income distribution across different job roles

📈 Power BI Visualizations Used

1.Bar Charts

2.Pie Charts

3.Donut Charts

4.Line Charts

6.Cards (KPI indicators)

7.Slicers for interactive filtering

⚙️ Tools & Technologies

1.Microsoft Power BI

2.Power Query for data cleaning and transformation

3.DAX (Data Analysis Expressions) for calculated measures

🚀 How to Use

1.Download the .pbix file from this repository.

2.Open it using Microsoft Power BI Desktop.

3.Explore the dashboard using filters and slicers to analyze HR data.

📊 Insights from the Dashboard

1.Identifies departments with high attrition rates

2.Shows employee distribution across various job roles

3.Highlights workforce demographics such as age and gender

📌 Conclusion

This HR Analysis Dashboard helps organizations understand workforce patterns and improve HR strategies through data-driven insights.
